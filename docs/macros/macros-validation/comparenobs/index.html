%compareNobs

Created by Jonathan Amato

Descriptionâ€‹

compareNobs creates a dataset that compares mismatching observation counts between two datasets by the specified variables. It also creates comparenobs_dataset1 and comparenobs_dataset2 datasets that are subsets of the original datasets (based on mismatches). This macro should be used whenever there is a mismatch on observation counts with the production programmer. It is not meant to be left in your code, so always call this macro while inside a comment block.

Syntaxâ€‹

/*
%macro compareNobs(dataset1, dataset2, vars, keep=_all_, debug=0);
*/
