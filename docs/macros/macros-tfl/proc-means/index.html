%proc_means

Created by Jonathan Amato

âœ… QC Version: %qc_proc_meansâ€‹

Created by Emily Johnson

Descriptionâ€‹

proc_means does three main things:

  1. Performs proc means
  2. Creates character values for each test based on the precision basis from %precision_basis
  3. Adjusts individual values (e.g. extending precision for "-0.0" values to the first nonzero number or setting STD to "N/A" when n=1)

Defintionâ€‹

%macro proc_means(
    data = null,
    by = null,
    var = null,
    out = null,
    debug = 0
);

dataâ€‹

This is the same value you would use in a normal proc means.

byâ€‹

This is the same value you would use in a normal proc means.

varâ€‹

This is the same value you would use in a normal proc means.

out or outputâ€‹

This is the same value you would use in a normal proc means.

Optional Parametersâ€‹

debugâ€‹

Set to 1 or Y if you want proc_means to preserve all datasets and variables created in the macro for debugging purposes.

Examplesâ€‹

Table summary statsâ€‹

** get precision_basis **;
%precision_basis(
    data = adlb,
    by   = paramcd param,
    var  = aval
);

** AVAL summary stats **;
%proc_means(
    data = adlb,
    by   = paramcd param avisitn avisit trt01an,
    var  = aval,
    out  = adlb_stats_aval
);

** CHG summary stats **;
%proc_means(
    data = adlb,
    by   = paramcd param avisitn avisit trt01an,
    var  = chg,
    out  = adlb_stats_chg
);

Overwriting precision basis for specific parametersâ€‹

** get precision_basis **;
%precision_basis(
    data = adlb,
    by   = paramcd param,
    var  = aval
);

** overwrite precision basis for specific parameters **;
data precision_basis;
    set precision_basis;
    
    if paramcd = "HDL" then precision_basis = 2;
    if paramcd = "MCH" then precision_basis = 1;
Run;

** AVAL summary stats **;
%proc_means(
    data = adlb,
    by   = paramcd param avisitn avisit trt01an,
    var  = aval,
    out  = adlb_stats_aval
);

** CHG summary stats **;
%proc_means(
    data = adlb,
    by   = paramcd param avisitn avisit trt01an,
    var  = chg,
    out  = adlb_stats_chg
);

Outcomeâ€‹

out datasetâ€‹

Dataset with character statistics.
