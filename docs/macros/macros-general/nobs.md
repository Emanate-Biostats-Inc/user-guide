# %nobs

## Description

**nobs** prints the number of observations for the given dataset.

## Defintion

```sas
%macro nobs(dataset);
```

## Required Parameters

### _dataset_

## Optional Parameters

None

## Example

```sas
%nobs(sdtm.vs);
```
