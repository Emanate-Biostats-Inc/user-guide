style_guide_checks checks your final TFL dataset for Style Guide rules and provided datasets and log reports. This is automatically called inside %rtfbreak for production table and listing programs. Validation TFL programmers can also call this if they want on their final datasets.

Set to part number. This is only used inside the %rtfbreak call.

debugâ€‹

Set to 1 if you want style_guide_checks to preserve all datasets created in the macro for debugging purposes.

Examplesâ€‹

Outcomeâ€‹

Log warnings for any style guide voilations.

Dataset that provides more detail for the footnote violations and expected fixes.
