# %progcheck

_Created by [**Kris Ilano**](mailto:kristoffer.ilano@emanatebiostats.com?subject=User%20Guide:%20progcheck)_

## Description

**progcheck** checks the log of an individual program for ERRORS, WARNINGS, and UNWANTED NOTES.

## Syntax

```sas
%macro progcheck;
```

## Required Parameters

None

## Optional Parameters

None

## Example

```sas
%progcheck;
```
