Saving Settings

Preferencesâ€‹

SAS has a counterintuitive way of saving some of your session settings. Instead of the more common Apply or Save, you have to follow these steps.

  1. Customize your window states to how you want them

  2. Tools > Options > Preferences...

The Preferences window will appear.

  1. Check Save settings on exit

  2. OK

  3. Exit SAS

  4. Start SAS

  5. Go back into Preferences and uncheck Save settings on exit

If you don't uncheck the box, your SAS will always save the current window states on exit.
